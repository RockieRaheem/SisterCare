## 🌸 The Problem We Solve

Millions of women and girls across Africa face silent struggles every day:

### 😔 **Mental Health in Silence**

- Many girls experience anxiety, depression, trauma, or emotional distress but have **no one to talk to**
- Cultural stigmas make it difficult to seek help or discuss personal problems
- Some have people around them but feel **too shy or embarrassed** to open up
- Access to professional mental health support is limited and expensive

### 🩸 **Menstrual Health Confusion**

- Many young women **don't know how to track their menstrual cycles**
- Without proper tracking, they face **unexpected periods** leading to:
  - Missing school or exams
  - Embarrassment at work or public places
  - Unpreparedness and anxiety
- Lack of education about menstrual phases, fertility windows, and health

### 🤝 **Need for Human Connection**

- Sometimes, an AI isn't enough — women need to talk to **real humans**
- Professional counsellors who are **strangers** can be easier to open up to than family
- Crisis situations require immediate human intervention

## 🤖 Meet Sister: An AI Agent, Not Just a Chatbot

**Sister is fundamentally different from traditional chatbots.**

### What Makes Sister an AI Agent?

| Traditional Chatbot  | Sister AI Agent                                                       |
| -------------------- | --------------------------------------------------------------------- |
| Responds to messages | **Takes actions** on your behalf                                      |
| Generic responses    | **Remembers your cycle** and personalizes responses                   |
| Can only chat        | **Logs symptoms**, **calculates fertility**, **searches health info** |
| No context awareness | **Understands your menstrual phase** and adjusts advice               |
| Static knowledge     | **Function calling** to access tools and external resources           |

### Sister's Intelligent Capabilities

**1. Function Calling & Tool Use**
Sister is powered by **Google Gemini 2.5 Flash-Lite** with function calling. This means she can:

- 📝 **Log your symptoms** directly to your health record
- 📊 **Retrieve your cycle data** and analyze patterns
- 🔍 **Search health information** from trusted databases
- 📅 **Calculate fertility windows** and ovulation dates
- 🏥 **Find healthcare resources** near you (Uganda-specific)
- 🚨 **Detect crisis situations** and provide emergency resources

**2. Contextual Intelligence**

- Knows which **phase of your cycle** you're in (menstrual, follicular, ovulation, luteal)
- Adjusts responses based on **your tracked symptoms**
- Provides **phase-specific advice** (e.g., iron-rich foods during period, self-care during PMS)

**3. Empathetic Companionship**

- Trained to be a **supportive, non-judgmental listener**
- Handles sensitive topics with care (abuse, harassment, depression)
- Guides users to professional help when AI support isn't enough

```
"Sister, I haven't told anyone this but... I've been feeling really down lately."

Sister: I hear you, and I'm so glad you felt safe enough to share this with me 💜
       Feeling low is never easy, especially when carrying it alone.
       Can you tell me more about what's been going on? I'm here to listen.

       [If you'd prefer to talk to a human counsellor, I can connect you with someone
       who specializes in mental wellness. Would that help?]
```

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn
- Firebase account
- Google AI Studio API key (for Gemini)

### Installation

1. **Clone the repository**

```bash
git clone https://github.com/RockieRaheem/SisterCare.git
cd SisterCare
```

2. **Install dependencies**

```bash
npm install
```

3. **Set up environment variables**

```bash
cp .env.example .env.local
```

Edit `.env.local` with your credentials:

```env
# Firebase Configuration
NEXT_PUBLIC_FIREBASE_API_KEY=your_api_key
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com
NEXT_PUBLIC_FIREBASE_PROJECT_ID=your_project_id
NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=your_project.appspot.com
NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
NEXT_PUBLIC_FIREBASE_APP_ID=your_app_id

# Gemini AI (required for chat)
GEMINI_API_KEY=your_gemini_api_key
```

4. **Run the development server**

```bash
npm run dev
```

5. **Open in browser**
   Navigate to [http://localhost:3000](http://localhost:3000)

## 🔧 Firebase Setup

1. Go to [Firebase Console](https://console.firebase.google.com/)
2. Create a new project called "SisterCare"
3. Enable Authentication:
   - Email/Password provider
   - Google Sign-In provider
4. Create a Firestore database
5. Copy configuration to `.env.local`

## 🤖 Gemini AI Setup

1. Go to [Google AI Studio](https://aistudio.google.com/)
2. Create an API key
3. Add to `.env.local` as `GEMINI_API_KEY`

The AI uses **Gemini 2.5 Flash-Lite** for:

- Fast response times
- Better rate limits on free tier
- Function calling for tool use (symptom logging, cycle info, etc.)
